📣 Marketing Funnel

MQL conversion rates by acquisition channel

🔍 Best Channel: Paid Search (12.3%)

🚚 Logistics & Delivery

Delivery performance, on-time rates, and satisfaction correlation

✅ 91.9% On-Time Delivery

Strong baseline performance. Each additional day of delivery time costs approximately 0.05 review points. Fast shipping = higher scores = more repeat buyers.

❌ 8.1% Late Deliveries

North and Northeast regions show the highest late rates (12–15%). Carrier SLA renegotiation + proactive SMS notification at T+12 days could recover 0.3 review points.

📦 Avg 12.1 Days to Deliver

Southeast customers receive in 8–10 days; North/Northeast in 16–22 days. A regional fulfilment centre would halve northern delivery times.

🗺️ Geographic Analysis

Order distribution and demand patterns across Brazilian states

🏙️ São Paulo Dominates

SP accounts for 42% of all orders — healthy penetration relative to its 22% population share. Strong logistics infrastructure to maintain.

🌱 North Region Opportunity

North region: 8.5% of Brazil's population but less than 1% of orders. Above-average spend per order suggests willingness to pay. Highest growth potential.

📦 Northeast Gap

Northeast: 27% of population, only ~8% of orders. Infrastructure and logistics challenges are the primary barrier — address with hub expansion before marketing spend.

💳 Payment Behaviour

Payment method distribution and instalment patterns

💳 Credit Card Dominates (74%)

3-instalment credit is the most popular choice, followed by 1-instalment. Instalment flexibility increases conversion — consider offering 10+ month options.

🏦 Boleto at 19%

Boleto (cash-in-advance bank slip) indicates ~20% of customers are unbanked or prefer offline payment. Vouchers (6%) are mostly loyalty/promo redemptions.

💡 BNPL Opportunity

High instalment usage (avg 3.7 for credit card orders) signals appetite for Buy Now Pay Later. Offering 12-month 0% interest for AOV > R$200 could increase basket size 25%.

⚙️ Pipeline Operations

How to run each stage of the Olist analytics platform from your laptop

✅ Prerequisites

Python 3.11+, pip install -r requirements.txt, .env file configured, CSV files in data/raw/

📁 Working Directory

All commands run from the olist_platform/ project root. Paths resolve automatically via .env.

⚠️ First-Time Setup

Copy .env.example → .env, fill in GCP_PROJECT_ID and GOOGLE_APPLICATION_CREDENTIALS before running ingestion.

1

Configure Environment

Copy example env file, fill in credentials. Required before any other step.

cp .env.example .env && nano .env
🔑
2

Place CSV Data Files

Copy all 11 Olist CSV files into the data/raw/ folder.

cp ~/Downloads/olist_*.csv data/raw/
📁
2b

Set Up BigQuery Datasets (first time only)

Creates olist_raw, olist_analytics, olist_analytics_staging, and olist_analytics_marts datasets.

bash scripts/setup_bigquery.sh
☁️
3

Data Quality — Core Datasets

Runs 51+ checks on orders, customers, products, etc. before loading.

python data_quality/dq_validation.py
🔍
4

Data Quality — Geolocation

Runs 30 checks on the 1M-row geolocation dataset.

python data_quality/dq_geolocation.py
🗺️
5

Ingest Core Datasets → BigQuery

Loads all 10 CSV files into BigQuery olist_raw dataset.

python ingestion/ingest_to_bigquery.py
📥
6

Ingest Geolocation → BigQuery

Deduplicates 1M rows to 19K zip prefixes and loads DimGeography.

python ingestion/ingest_geolocation.py
📍
7

Create Warehouse Schema

Run schema.sql in BigQuery console to create all dimension and fact tables.

bq query --use_legacy_sql=false < warehouse/schema.sql
🏛️
8

DBT Transformations

Runs all 9 staging models + 7 mart models, then runs 80+ schema tests and 6 SQL integration tests.

cd dbt_project && dbt deps && dbt run && dbt test
🔄
8b

Generate DBT Lineage Documentation (optional)

Produces an interactive DAG showing all model dependencies. Browse at localhost:8080.

cd dbt_project && dbt docs generate && dbt docs serve
🔗
9

Generate Analysis & Charts

Produces all 8 charts to reports/charts/. Open as Jupyter notebook for interactive exploration.

python notebooks/olist_analysis.py
📊
10

Open This Dashboard

Open dashboard/index.html in any browser. No server needed — fully self-contained.

open dashboard/index.html
